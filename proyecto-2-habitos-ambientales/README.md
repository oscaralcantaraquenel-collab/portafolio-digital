# Aplicación Educativa de Hábitos de Cuidado Ambiental

## Descripción
Aplicación móvil enfocada en la educación ambiental dentro del sector escolar. 
Permite a los profesores crear clases, asignar retos y compartir contenido 
educativo sobre el cuidado del medio ambiente. Los alumnos pueden consultar 
materiales, completar tareas y retos, y acumular puntos que sirven como base 
para su evaluación.

## Contexto del proyecto
Desarrollado como proyecto integrador del **10º cuatrimestre**, con una 
duración aproximada de **8 semanas**.

## Tecnologías utilizadas
- **IDE:** Android Studio
- **Framework:** Flutter
- **Lenguaje:** Dart
- **Base de datos y backend:** Firebase (autenticación, almacenamiento de 
  archivos y gestión de registros de profesores y alumnos)

## Funcionalidades principales

**Para profesores:**
- Creación de clases y talleres
- Asignación de retos y contenido educativo
- Subida de materiales de aprendizaje

**Para alumnos:**
- Visualización de materiales de clase
- Acceso a contenido adicional subido por otros usuarios
- Entrega de tareas y retos
- Sistema de puntos como apoyo para la calificación de entregas

## Público objetivo
Instituciones educativas con acceso a Internet, interesadas en impartir 
talleres o cursos sobre cuidado ambiental de forma digital e interactiva.

## Requisitos
Esta aplicación requiere conexión a Internet para su funcionamiento.
