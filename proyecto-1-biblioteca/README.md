# Sistema de Administración de Bibliotecas Escolares

## Descripción
Aplicación móvil de escritorio local para la gestión de bibliotecas escolares, 
pensada especialmente para instituciones de bajos recursos sin acceso a Internet. 
Permite a un único administrador gestionar de forma sencilla los préstamos de 
libros, espacios (salones, salas) y equipo de cómputo, todo desde una interfaz 
intuitiva y fácil de usar.

## Contexto del proyecto
Desarrollado como proyecto integrador del **5º cuatrimestre**, con una duración 
aproximada de **8 semanas**.

## Tecnologías utilizadas
- **IDE:** Visual Studio Community 2022
- **Framework:** Xamarin
- **Lenguajes:** Kotlin
- **Base de datos:** SQLite (almacenamiento local, sin necesidad de conexión a Internet)
- **Arquitectura:** MVC (Modelo - Vista - Controlador)

## Funcionalidades principales
- Registro y control de préstamos de libros
- Administración de préstamos de espacios (salones, salas de estudio, etc.)
- Gestión de préstamos de equipo de cómputo
- Funcionamiento 100% local, sin dependencia de Internet
- Interfaz sencilla e intuitiva pensada para un solo administrador

## Público objetivo
Escuelas con recursos limitados y sin acceso constante a Internet, que buscan 
una forma sencilla de administrar sus recursos internos.
