# Portafolio Digital

Portafolio de proyectos de software desarrollados durante mi formación en 
Software Development and Management.

## Proyectos

### [Sistema de Administración de Bibliotecas Escolares](./proyecto-1-biblioteca)
Aplicación móvil local con Xamarin y SQLite para la gestión de préstamos 
de libros, espacios y equipo en escuelas sin acceso a Internet.

### [Aplicación Educativa de Hábitos de Cuidado Ambiental](./proyecto-2-habitos-ambientales)
Aplicación móvil desarrollada en Flutter con Firebase, enfocada en la 
educación ambiental para profesores y alumnos.
